# Basic English Grammar Game

A standalone beginner grammar game for students.

Current module:

- Lesson 01: 分辨句子是否有動詞
- Students answer with Tick / Cross.
- Verb sentences reveal an English sentence with browser speech playback.
- No-verb sentences ask whether `is / am / are` is needed, then ask students to pick the correct be verb.
